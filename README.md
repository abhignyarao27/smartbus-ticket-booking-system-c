# SmartBus Ticket Booking System (C)

## Description
The SmartBus Ticket Booking System is a console-based application developed using the C programming language.
It allows users to select bus routes, choose available travel timings, enter ticket details, and calculate the total fare.
The program follows a simple menu-driven approach.

## Features
- Displays available bus routes
- Shows multiple travel timings for each route
- Allows ticket quantity selection
- Calculates total ticket fare
- Simple and user-friendly console interface

## Routes Available
1. Razole to Vijayawada
2. Hyderabad to Bangalore
3. Kakinada to Chennai
4. Amalapuram to Eastpallem

## Technologies Used
- C Programming Language

## How to Run
1. Compile the program using any C compiler
2. Run the executable file
3. Follow the on-screen instructions
