# SmartBus Ticket Booking System (C)

## Description
The SmartBus Ticket Booking System is a console-based application developed using the C programming language.
It allows users to select bus routes, choose available travel timings, enter ticket details, and calculate the total fare
through a simple menu-driven interface.

## Working Principle
The system displays available bus routes and timings to the user.
After selecting a route and preferred time, the user enters the number of tickets.
The program calculates the total ticket cost based on the selected route and displays the booking details.

## Algorithm
1. Start the program
2. Display available bus routes and prices
3. Accept the user’s route choice
4. Display available timings for the selected route
5. Accept the time selection
6. Accept the number of tickets
7. Calculate total fare
8. Display journey details and total amount
9. End the program

## Pseudocode
START  
Display routes and prices  
Read route choice  

IF route choice is invalid  
  Display error and exit  
ENDIF  

Display timings for selected route  
Read time choice  

IF time choice is invalid  
  Display error and exit  
ENDIF  

Read number of tickets  
Calculate total amount  

Display journey details  
Display total fare  
END
